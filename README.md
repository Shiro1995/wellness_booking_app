# Wellness Booking App

This is a simple repository for building a mobile app by React Native framework. 

## How To Use?

After cloning the repository, get all project-dependencies by running the command above at terminal:
```
 yarn install
```
Then run app on the simulator or your real-device.



**for iOS:**
```
 yarn ios
```
**for Android:**
```
 yarn android
```

## Tech-Stack
- React-Navigation
- Redux, Redux-saga, Redux-toolkit
- Javascript
